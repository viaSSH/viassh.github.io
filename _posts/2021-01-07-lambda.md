##### **Show Tags** by using AWS CLI

**Input**

```
aws lambda list-tags --resource arn:aws:lambda:ap-northeast-2:000000000000:function:getDeviceLog
```

**output**

```json
{
    "Tags": {
        "iot": "device-log"
    }
}
```
